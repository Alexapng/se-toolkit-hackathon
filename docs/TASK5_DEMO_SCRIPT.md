# Task 5 Demo Script (about 2 minutes)

## 0:00 - 0:15

"This is Minimal Habit Bot, a Telegram Mini App for daily habit consistency.  
My target users are people who want to build small daily habits and keep momentum."

## 0:15 - 0:35

"I open the bot, send `/start`, and launch the Mini App.  
The profile is linked automatically using my Telegram username, so I do not need to enter a manual username."

## 0:35 - 0:55

"Now I create habits, for example `Drink water` and `Walk 10 minutes`.  
Each habit appears in my daily list with its current status."

## 0:55 - 1:20

"I press `Check in` to mark completed habits for today.  
The dashboard updates completed count, total habits, and streak information for my selected date."

## 1:20 - 1:40

"When all habits are complete, I get the motivational streak message:  
`Yay! Great job!` and the current streak in days."

## 1:40 - 1:55

"I can also delete habits with one click.  
The UI updates immediately, and the app remains responsive."

## 1:55 - 2:00

"In Telegram, reminder commands like `/notify_on` and `/streak` support daily accountability.  
So Version 2 extends Version 1 with streaks, Telegram integration, and reminders."
